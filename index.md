# Em Cada Voto Igualdade

Bem-vindo ao **Em Cada Voto Igualdade**. Este é um projeto cívico, independente e apartidário que nasce com um propósito claro: debater e propor soluções para que **todos os votos dos portugueses passem a contar igualmente** para a eleição dos seus representantes.

O nosso sistema eleitoral atual faz com que centenas de milhares de votos sejam literalmente desperdiçados todos os anos, criando assimetrias graves onde o "custo" de eleger um deputado varia drasticamente consoante o círculo eleitoral ou o partido político.
